# Calculadora KPI

**Calculadora KPI** es una aplicación web diseñada para calcular indicadores clave de rendimiento (KPI) de forma sencilla, directa desde el navegador.

## Características principales

- Calculadora interactiva para diferentes KPIs.
- Permite ingresar datos relevantes y obtener resultados inmediatos.
- Visualización amigable y accesible.
- No requiere instalación ni dependencias adicionales.

## Estructura del repositorio

```
calculadora-kpi/
├── index.html      # Página principal y única archivo de la calculadora
└── README.md       # Documentación del proyecto
```

## Uso

1. Abre el archivo `index.html` en tu navegador.
2. Ingresa los valores requeridos para el cálculo del KPI de tu interés.
3. Obtén el resultado de forma instantánea.
